Graph formal definition

  • Graph \(G=(V,E)\), where \(V\) is a set of nodes and \(E\) is a set of edges.
  • An edge is defined as \({v,w}\), where \(v,w\in V\)
  • The order of a graph = the number of vertices = \(|G|\)
  • The size of a graph = the number of edges = \(\left\|G\right\|\)
  • The degree of node \(deg(v)\) is the number of edges incident to it
  • The geodesic distance between \(v\) and \(w\) = number of edges in the geodesic path between \(v\) and \(w\) = \(\delta(v,w)\)

centralities

  • B = Closeness centrality,

  • C = Betweenness centrality,
